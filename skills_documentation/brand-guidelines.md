# Brand Guidelines Skill

## Functionality

The `brand-guidelines` skill is designed to apply a consistent and professional brand identity to various artifacts. It provides a set of pre-defined brand colors and typography rules that can be used to style presentations, documents, and other visual materials.

## Workflow

The skill provides a set of brand guidelines that the agent should follow when styling artifacts. This includes:

*   **Colors:** A pre-defined palette of main and accent colors.
*   **Typography:** Specific fonts for headings and body text, with fallbacks for systems where the fonts are not available.
*   **Application:** The skill includes logic for applying these styles to text and shapes, ensuring a consistent and on-brand look and feel.

## Authentication

The `brand-guidelines` skill does not have any direct authentication requirements. It provides a set of styling rules and does not interact with any external services that require API keys or tokens. However, the skill is designed to be used in conjunction with other tools (e.g., `python-pptx`), which may have their own dependencies or requirements.
