# Frontend Design Skill

## Functionality

The `frontend-design` skill is designed to create distinctive, production-grade frontend interfaces with a high level of design quality. It guides the agent in building web components, pages, and applications that are not only functional but also visually striking and memorable. The skill emphasizes a "design thinking" approach, encouraging the agent to make bold aesthetic choices and avoid generic, AI-generated designs.

## Workflow

The workflow for this skill is centered around a "design thinking" process:

1.  **Understand the Context:** Before writing any code, the agent should understand the purpose of the interface, its audience, and any technical constraints.

2.  **Choose an Aesthetic Direction:** The agent is encouraged to commit to a bold and clear aesthetic direction, such as minimalism, maximalism, or a retro-futuristic theme.

3.  **Implement with High Quality:** The agent then implements the design using standard web technologies (HTML/CSS/JS, React, Vue, etc.), with a focus on:
    *   **Typography:** Using unique and interesting fonts.
    *   **Color & Theme:** Creating a cohesive and impactful color scheme.
    *   **Motion:** Incorporating animations and micro-interactions.
    *   **Spatial Composition:** Using creative and unexpected layouts.

## Authentication

The `frontend-design` skill does not have any direct authentication requirements. It is a creative and technical skill that focuses on generating frontend code and does not interact with any external services that require API keys or tokens. However, the resulting frontend application may itself need to interact with authenticated APIs.
