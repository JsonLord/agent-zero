# Figures and Tables

Figures and tables are used to present data in a clear and concise manner.

## 1. Figures

- Used to present data in a graphical format.
- Should be clear, easy to read, and relevant to the text.
- Each figure should have a caption that explains the figure and its components.

## 2. Tables

- Used to present data in a tabular format.
- Should be clear, easy to read, and relevant to the text.
- Each table should have a title that explains the table and its contents.
